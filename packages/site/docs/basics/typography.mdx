# Typography

## TypeScale

A **type scale** is a selection of font styles that can be used across an app, ensuring a flexible, yet consistent, style that accommodates a range of purposes. The Material Design type scale is a combination of 15 styles, each with an intended application and meaning. They’re assigned based on use (such as display or headline), and grouped more broadly into categories based on scale (such as large or small). Default type scale uses Roboto for all titles, labels, and body text, creating a cohesive typography experience.

Typescale distinguishes between two types of reference tokens: **brand** and **plain**.
This enables you to use two different typefaces within your design.

The **brand typeface** (which has the token ds.ref.typeface.brand) is recommended for larger font sizes, such as Headline and Display,
while **plain typeface** (with the token ds.ref.typeface.plain) is suitable for smaller font sizes, such as Body and Label.

:::tip
See [typeface tokens](../reference-tokens#typefaces).
:::
